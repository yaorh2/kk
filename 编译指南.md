# K线训练APP - 完整编译指南

## 📋 前置要求

### 必需软件
1. **Android Studio** - 最新版本（推荐 Hedgehog / Iguana）
   - 下载地址：https://developer.android.com/studio

2. **JDK 11+** - Android Studio已内置

3. **Android SDK** - API 34 (Android 14)

---

## 🚀 方法一：使用Android Studio编译（推荐）

### 步骤1：打开项目
1. 启动Android Studio
2. 选择 `Open an Existing Project`
3. 选择 `K线训练APP/android_app` 目录
4. 等待Gradle同步完成（首次打开可能需要5-10分钟）

### 步骤2：编译APK
```
菜单栏 → Build → Build Bundle(s) / APK(s) → Build APK(s)
```

### 步骤3：安装到手机
编译完成后，APK位置：
```
K线训练APP/android_app/app/build/outputs/apk/debug/app-debug.apk
```

安装方式：
- 直接将APK文件发送到手机安装
- 或使用ADB命令：
  ```bash
  adb install app/build/outputs/apk/debug/app-debug.apk
  ```

## ⚠️ 常见问题解决

### Q1: Gradle同步失败
**解决：**
1. 检查网络连接（需要下载依赖）
2. 配置国内镜像源：
   - 阿里云：`maven { url 'https://maven.aliyun.com/repository/google' }`
   - 腾讯云：`maven { url 'https://mirrors.cloud.tencent.com/nexus/repository/maven-public/' }`

### Q2: SDK版本不匹配
**解决：**
打开 `File → Project Structure → SDK Location`
确认Android SDK路径配置正确

### Q3: 编译报错：compileSdkVersion已弃用
**解决：**
项目已使用新的API `compileSdk 34`，请使用Android Studio Gradle Plugin 8.0+

### Q4: APK安装失败
**解决：**
1. 手机开启"允许安装未知来源应用"
2. 检查手机Android版本 ≥ 7.0
3. 卸载旧版本后重新安装

## 🔄 获取更多股票数据

### 使用Python脚本获取A股数据
```bash
cd K线训练APP/python_scripts
pip install -r requirements.txt

# 获取示例数据（推荐）
python stock_data_fetcher.py --mode sample

# 获取全市场数据（耗时较长）
python stock_data_fetcher.py --mode all

# 增量更新数据
python stock_data_fetcher.py --mode update
```

生成的JSON数据文件位于 `K线训练APP/python_scripts/output/` 目录，可通过APP的"数据管理"功能导入。
