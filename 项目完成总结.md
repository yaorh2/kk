## 🚀 使用方法

### 快速体验

1. **编译安装**
   ```bash
   cd K线训练APP/android_app
   ./gradlew assembleDebug
   adb install app/build/outputs/apk/debug/app-debug.apk
   ```

2. **开始训练**
   - 打开应用，点击"开始训练"
   - 系统自动加载示例数据开始训练

### 获取真实数据

```bash
cd K线训练APP/python_scripts
pip install -r requirements.txt
python stock_data_fetcher.py --mode sample
```

将生成的JSON文件导入手机即可使用真实A股数据训练。
