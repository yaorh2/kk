## Android应用源码详解

### 1. Application 类

**KlineTrainingApp.kt**
- 应用程序入口
- 初始化全局单例
- 持有DataManager实例

### 2. 数据层 (data/)

#### KlineData.kt - 数据模型
```kotlin
// K线数据
data class KlineData(
    val date: String,
    val open: Float,
    val close: Float,
    val high: Float,
    val low: Float,
    val volume: Long
)

// 股票完整数据
data class StockData(
    val code: String,
    val name: String,
    val market: String,
    val total_days: Int,
    val start_date: String,
    val end_date: String,
    val data: List<KlineData>
)

// 交易记录
data class TradeRecord(...)

// 技术指标数据
data class TechnicalIndicators(...)

// 训练会话
data class TrainingSession(...)

// 统计数据
data class StatisticsData(...)
```

#### DataManager.kt - 数据管理
- 加载/保存股票数据
- 管理训练会话历史
- 计算统计数据
- 支持JSON数据导入

主要功能：
```kotlin
// 获取可用股票列表
suspend fun getAvailableStocks(): List<StockData>

// 随机选择股票
suspend fun getRandomStock(): StockData?

// 导入外部数据
suspend fun importStockData(json: String): Boolean

// 保存训练会话
suspend fun saveTrainingSession(session: TrainingSession)

// 获取统计数据
suspend fun getStatistics(): StatisticsData
```

#### IndicatorCalculator.kt - 技术指标计算
计算各类技术指标：

**移动平均线 (MA)**
- MA5、MA10、MA20
- 简单移动平均算法

**MACD**
- DIF线：EMA12 - EMA26
- DEA线：DIF的EMA9
- MACD柱状图：2 * (DIF - DEA)

**KDJ**
- K值：RSV的平滑
- D值：K值的平滑
- J值：3K - 2D

**RSI**
- 相对强弱指标
- 默认周期14天

### 3. UI层 (ui/)

#### MainActivity.kt - 主界面
- 显示应用标题
- 展示训练概览（数据数量、训练次数、胜率）
- 入口按钮：
  - 开始训练
  - 训练统计
  - 数据管理

#### TrainingActivity.kt - 训练界面
核心训练逻辑：

```kotlin
// 训练流程
1. 随机选择股票和起始位置
2. 初始化资金、持仓
3. 显示K线图（到当前位置为止）
4. 用户执行买卖操作
5. 点击"下一根K线"推进训练
6. 结束训练，保存记录，显示结果
```

核心功能：
- K线图展示 + 技术指标
- 买入/卖出操作
- 仓位选择
- 快速模式（自动推进）
- 实时资金/持仓显示
- 交易标记在K线图上显示

#### StatisticsActivity.kt - 统计界面
- 总体统计：训练次数、总交易数、胜率、平均收益率
- 极值统计：最大回撤、最佳收益、最差收益
- 历史记录列表：每次训练的详细信息

#### DataManageActivity.kt - 数据管理
- 显示当前数据统计
- 导入外部JSON数据
- 使用说明
- 调用系统文件选择器

#### KlineChartView.kt - 自定义K线图
```kotlin
// 绘制内容
1. K线实体 + 影线
   - 阳线：红色
   - 阴线：绿色

2. 移动平均线
   - MA5: 橙色
   - MA10: 蓝色
   - MA20: 紫色

3. 成交量柱状图
   - 显示在K线图下方

4. MACD指标
```

支持的手势：
- 横向滑动查看历史K线
- 双指缩放调整时间范围

## Python数据获取脚本详解

### stock_data_fetcher.py

**功能**：
- 使用AKShare开源库获取A股历史数据
- 支持数据清洗和格式统一
- 增量更新功能
- 按月/按股票分文件存储

**主要类和函数**：

```python
# 加载/保存配置
load_config() / save_config()

# 获取A股股票列表
get_stock_list()

# 获取单只股票历史数据
get_stock_history(code, market, start_date, end_date)

# 保存为JSON格式
save_stock_data(code, name, df, market)

# 增量更新
incremental_update(code, name, market)

# 批量获取
fetch_all_stocks(limit=100)

# 创建示例数据集
create_sample_dataset()
```

**使用方法**：
```bash
# 获取示例数据（推荐）
python stock_data_fetcher.py --mode sample

# 获取100只股票
python stock_data_fetcher.py --mode all --limit 100

# 更新已有数据
python stock_data_fetcher.py --mode update
```

**数据来源**：
- AKShare支持的数据源
- 新浪财经、腾讯财经等
- 包含前复权价格

## 数据格式规范

### 股票JSON格式
```json
{
  "code": "000001",           // 股票代码
  "name": "平安银行",          // 股票名称
  "market": "SZ",             // 市场：SH(上海), SZ(深圳)
  "total_days": 100,          // 数据总天数
  "start_date": "2023-01-01", // 起始日期
  "end_date": "2023-05-20",   // 结束日期
  "data": [
    {
      "date": "2023-01-03",   // 日期
      "open": 12.50,          // 开盘价
      "close": 12.80,         // 收盘价
      "high": 12.90,          // 最高价
      "low": 12.45,           // 最低价
      "volume": 52340000      // 成交量（股）
    }
  ]
}
```

### 文件命名规范
- 格式：`{股票代码}_{市场}.json`
- 示例：`000001_SZ.json`, `600519_SH.json`

## 编译和运行

### 使用Android Studio
1. 打开 `android_app/` 目录
2. 等待Gradle同步完成
3. 连接Android设备或启动模拟器
4. 点击"Run"按钮（绿色三角形）

### 使用命令行
```bash
cd android_app

# Debug版本
./gradlew assembleDebug
# 输出: app/build/outputs/apk/debug/app-debug.apk

# Release版本
./gradlew assembleRelease
# 输出: app/build/outputs/apk/release/app-release.apk
```
