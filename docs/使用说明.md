## 安装说明

### 系统要求
- Android 7.0 (API 24) 或更高版本
- 推荐使用 Android 10 及以上版本以获得最佳体验
- 屏幕分辨率建议 1080p 或更高

### 安装步骤

#### 方法一：使用APK文件安装
1. 下载 `K线训练_v1.0.0.apk` 文件到手机
2. 打开手机"设置" → "安全" → 允许"未知来源"安装
3. 使用文件管理器找到APK文件并点击安装
4. 安装完成后点击"打开"启动应用

#### 方法二：使用Android Studio编译安装
1. 将项目导入Android Studio
2. 连接Android设备并开启USB调试
3. 点击"Run"按钮编译并安装到设备

## 快速开始

### 第一次使用

1. **启动应用**
   - 打开K线训练APP
   - 应用会自动加载内置示例数据

2. **查看主页信息**
   - 查看当前股票数据数量
   - 查看历史训练次数和胜率

3. **开始第一次训练**
   - 点击"开始训练"按钮
   - 系统随机选择一只股票开始训练
   - 进入横屏训练界面

### 基础操作

**查看K线**
- 横向滑动可查看历史K线
- 双指捏合可缩放K线图

**推进训练**
- 点击"下一根K线"按钮显示下一根K线
- 或点击"快速模式"自动推进

**完成交易**
- 选择仓位（全仓/半仓/1/4仓）
- 点击"买入"或"卖出"按钮
- 交易后K线图上会显示箭头标记

## 训练模式详解

### 训练原理

K线训练模式模拟真实交易场景：
- **隐藏股票信息**：训练时不显示股票名称和代码，避免心理暗示
- **逐根显示**：只能看到到当前为止的K线，无法预知未来
- **实时决策**：每看到新的K线都可以做出买卖决策
- **结果统计**：训练结束后显示详细的交易记录和收益情况

### 技术指标说明

**移动平均线 (MA)**
- MA5：5日均线，显示短期趋势
- MA10：10日均线，显示中期趋势  
- MA20：20日均线，显示长期趋势
- 颜色：MA5(橙)、MA10(蓝)、MA20(紫)

**成交量 (Volume)**
- 显示在K线图下方
- 阳线红色、阴线绿色

**MACD指标**
- DIF线（黄色）：快速移动平均线
- DEA线（青色）：慢速移动平均线
- MACD柱状图：金叉/死叉信号

## 模拟交易指南

### 交易规则

**初始资金**
- 默认初始资金：100,000元
- 可以在设置中修改（开发中）

**交易单位**
- 股票交易以100股为1手
- 买入数量必须是100的整数倍
- 卖出数量可以是任意整数

**交易价格**
- 买入/卖出价格按当天收盘价计算
- 不考虑交易手续费和滑点
- 不考虑涨跌停限制

### 仓位管理

**全仓模式**
- 使用所有可用资金买入
- 适合强烈看涨时使用

**半仓模式**
- 使用50%可用资金买入
- 平衡风险和收益

**1/4仓模式**
- 使用25%可用资金买入
- 保守交易，控制风险

### 交易策略建议

1. **顺势交易**
   - 上升趋势中以做多为主
   - 下降趋势中以观望或做空为主（如支持）

2. **止损止盈**
   - 设定合理的止损位，通常3-5%
   - 达到预期收益时及时止盈

3. **仓位控制**
   - 不要一次性满仓
   - 分批建仓，分散风险

4. **指标结合**
   - 结合MA、MACD、成交量等多个指标
   - 单一指标信号不可靠

## 数据管理

### 数据获取

#### 使用Python脚本获取真实数据

1. **安装依赖**
```bash
cd python_scripts
pip install -r requirements.txt
```

2. **获取数据**
```bash
# 获取示例数据（推荐首次使用）
python stock_data_fetcher.py --mode sample

# 获取100只股票数据
python stock_data_fetcher.py --mode all --limit 100

# 更新已有数据
python stock_data_fetcher.py --mode update
```

3. **脚本功能说明**
   - 使用AKShare开源库获取A股历史数据
   - 数据来源：新浪财经、腾讯财经等
   - 包含：开盘价、收盘价、最高价、最低价、成交量
   - 自动按股票代码分文件存储
   - 支持增量更新

### 数据导入

1. **准备数据文件**
   - 确保数据为JSON格式
   - 参照示例数据格式
   - 建议单只股票数据不少于60个交易日

2. **导入步骤**
   - 将JSON文件复制到手机存储
   - 在应用主页点击"数据管理"
   - 点击"导入外部数据"按钮
   - 在文件选择器中选择JSON文件
   - 等待导入完成提示

### 数据格式说明

```json
{
  "code": "股票代码",
  "name": "股票名称",
  "market": "市场(SH/SZ)",
  "total_days": 总天数,
  "start_date": "起始日期",
  "end_date": "结束日期",
  "data": [
    {
      "date": "日期",
      "open": 开盘价,
      "close": 收盘价,
      "high": 最高价,
      "low": 最低价,
      "volume": 成交量
    }
  ]
}
```

## 常见问题

### 安装和启动

**Q: 安装时提示"解析包错误"怎么办？**
A: 请检查：
- Android版本是否满足要求（7.0+）
- APK文件是否完整下载
- 存储空间是否充足

**Q: 应用启动后闪退？**
A: 请尝试：
- 重启应用
- 清除应用缓存
- 重新安装APK

### 数据相关

**Q: 提示"数据不足，请先导入数据"？**
A: 应用需要至少60个交易日的K线数据才能开始训练。
- 可以使用提供的Python脚本获取数据
- 或导入自己的JSON数据文件

**Q: 数据导入失败怎么办？**
A: 请检查：
- JSON文件格式是否正确
- 文件是否损坏
- 是否符合规定的数据结构
- 单只股票数据是否足够（至少60条）

### 训练相关

**Q: K线图显示不完整？**
A: 请检查：
- 手机是否为横屏模式
- 尝试重启训练
- 更新到最新版本

**Q: 无法进行交易？**
A: 请确认：
- 买入时是否有足够资金
- 卖出时是否有持仓
- 是否选择了仓位

### 其他

**Q: 训练记录会保存吗？**
A: 是的，每次训练完成后：
- 收益率、胜率等统计数据会保存
- 交易记录会保存到历史记录
- 可以在"训练统计"页面查看历史记录

**Q: 支持导出交易记录吗？**
A: 当前版本暂不支持，后续版本会添加导出功能。

**Q: 可以自定义初始资金吗？**
A: 当前版本固定为100,000元，后续版本会支持自定义设置。
