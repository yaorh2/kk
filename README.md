# K线训练APP - 专业股票技术分析训练工具

## 📱 产品简介

这是一款专为股票投资者设计的K线训练工具，通过模拟真实市场环境，帮助用户在无风险的情况下磨练技术分析能力、提升盘感、验证交易策略。

---

## ✨ 核心功能

### 📊 K线图展示
- **专业K线渲染**：蜡烛图、成交量柱、均线系统
- **技术指标**：MA5/MA10/MA20、MACD、KDJ、RSI
- **手势操作**：缩放、滑动、长按查看详情

### 🎮 训练模式
- **盲训模式**：隐藏股票名称和日期，纯技术分析
- **逐根推进**：一根一根K线推进，模拟真实交易
- **快速模式**：自动推进，适合快速浏览
- **随机选股**：从本地数据库随机选择股票

### 💰 模拟交易
- **买卖操作**：一键买入/卖出
- **仓位管理**：全仓、半仓、1/4仓灵活选择
- **交易记录**：所有交易标记在K线图上
- **盈亏统计**：实时计算盈亏和收益率

### 📈 数据统计
- **训练历史**：完整记录每次训练
- **胜率分析**：统计交易成功率
- **收益率曲线**：查看收益变化
- **最大回撤**：风险控制能力评估

### 🌐 **内置数据获取（新增）**
- **一键获取**：APP内置15只热门A股数据
- **在线更新**：直接从互联网免费API获取数据
- **多数据源**：新浪财经、东方财富等多个源自动切换
- **自动更新**：每月1号自动后台更新最新数据
- **单股查询**：输入股票代码单独获取

## 🛠️ 技术架构

### 前端技术栈
- **语言**：Kotlin
- **UI框架**：Android原生 + Material Design
- **图表库**：MPAndroidChart
- **网络请求**：OkHttp
- **数据存储**：JSON文件 + Room
- **后台任务**：WorkManager

### 数据源
- **新浪财经API**：免费K线数据
- **东方财富API**：备用数据源
- **腾讯财经API**：备用数据源
- **自动切换**：数据源失效时自动尝试下一个

## 🚀 快速开始

### 方法一：GitHub云编译（推荐）
1. 将整个项目上传到GitHub
2. 等待2-3分钟自动编译
3. 在Actions页面下载APK
4. 安装到手机即可使用

### 方法二：本地编译
1. 安装Android Studio
2. 打开 `android_app` 目录
3. 等待Gradle同步完成
4. Build → Build APK(s)
5. 安装生成的APK

## 📖 使用教程

### 第一次使用
1. 安装APP后打开
2. 点击"数据管理"
3. 点击"一键获取15只热门股票数据"
4. 等待数据下载完成
5. 返回首页，点击"开始训练"

### 训练模式使用
1. **开始训练**：系统随机选择一只股票，隐藏名称
2. **查看K线**：根据历史K线和指标分析走势
3. **买卖操作**：点击买入/卖出按钮进行交易
4. **推进K线**：点击"下一根"查看后续走势
5. **结束训练**：查看本次训练的盈亏统计

### 数据管理
- **一键获取**：快速获取15只热门A股数据
- **单股查询**：输入股票代码单独获取
- **自动更新**：开启后每月自动后台更新
- **数据统计**：查看本地股票数量和更新时间
